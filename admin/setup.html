🚀 Admin System Setup

Complete setup for the SkinCare Admin Panel with role-based access control

1. Customer Status System
Adds customer account status tracking (active, warned, banned) to enable customer management and ban functionality.
2. Admin Role System
Updates admin table with role-based permissions (Super Admin, Manager, Staff, Expert) and adds proper ENUM constraints.
3. Demo Admin Accounts
Creates demo admin accounts for each role type with default credentials for testing and development.
